## 项目简介

这是一个示例项目，展示如何使用 projects 内容集合创建项目展示页面。

## 主要功能

- 项目卡片展示
- 状态徽章（计划中、进行中、已发布等）
- 技术栈标签
- GitHub 和 Demo 链接

## 技术实现

项目使用 Astro Content Collections API 实现，通过 Zod schema 进行数据校验。
